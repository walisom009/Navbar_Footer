🚀 Vocação Plus 
Bem-vindo ao esqueleto oficial do Vocação Plus. Este projeto foi desenvolvido para servir como uma base sólida, elegante e profissional para uma plataforma de orientação vocacional e análise de carreira impulsionada por IA.

Atualmente, o projeto foca em uma estrutura minimalista e premium, priorizando a identidade visual da marca através de um cabeçalho e rodapé unificados.

🎨 Identidade Visual e Design
O design foi concebido para transmitir confiança e clareza.

Cor Principal: Azul Safira Médio (#003366), escolhido por equilibrar seriedade acadêmica com modernidade tecnológica.

Tipografia: Plus Jakarta Sans — uma fonte geométrica moderna que garante excelente legibilidade.

Layout: Estilo "Frame", onde o topo e a base emolduram o conteúdo principal, mantendo o foco do usuário no que realmente importa.

🛠️ Tecnologias Utilizadas
Para garantir performance e responsividade, utilizamos o que há de melhor no desenvolvimento front-end atual:

HTML5 & CSS3: Estrutura e estilização customizada.

Bootstrap 5: Framework para garantir que o site seja totalmente responsivo (mobile-friendly).

FontAwesome 6: Biblioteca de ícones profissionais.

Google Fonts: Integração de fontes premium.
